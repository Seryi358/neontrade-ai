# Creación y configuración de billeteras

**Duración:** 16:19

## Descripción

Configurar correctamente tus billeteras digitales es el primer paso para moverte con seguridad y autonomía en el mundo cripto. En este vídeo aprenderás a crear y proteger billeteras calientes como MetaMask y Phantom, así como a añadir redes y gestionar múltiples activos.

🔹 Puntos clave:
✔️ Paso a paso para descargar, instalar y configurar MetaMask y Phantom
✔️ Cómo generar y proteger tu seed phrase de forma segura
✔️ Añadir redes como Arbitrum, Polygon, Base o Mantle de forma manual
✔️ Transferencia de activos entre redes y uso de Chainlist.org
✔️ Diferencias entre redes: velocidad, comisiones y liquidez

Este conocimiento te dará el control total sobre tus criptoactivos y te permitirá interactuar con el ecosistema DeFi de forma segura y eficiente.

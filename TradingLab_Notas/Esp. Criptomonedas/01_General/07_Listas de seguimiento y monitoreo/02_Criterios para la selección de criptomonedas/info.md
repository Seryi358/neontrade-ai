# Criterios para la selección de criptomonedas

**Duración:** 06:55

## Descripción

Con miles de criptomonedas en el mercado, seleccionar las adecuadas para invertir o hacer trading puede ser abrumador. En este vídeo, aprenderás criterios clave para filtrar activos y construir una lista de seguimiento efectiva.

🔹 Puntos clave:
✔️ Cómo clasificar criptomonedas según su capitalización de mercado y volatilidad
✔️ Diferencias entre activos de alta, media y baja capitalización
✔️ Riesgos de operar con criptos poco líquidas y altamente manipulables
✔️ Cuántas criptomonedas incluir en tu lista para optimizar el análisis

Elegir correctamente en qué activos enfocarte te dará mayor claridad y control en tu operativa.

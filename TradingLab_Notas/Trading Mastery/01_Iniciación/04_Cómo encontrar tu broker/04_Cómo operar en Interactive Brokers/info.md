# Cómo operar en Interactive Brokers

**Duración:** 27:21

## Descripción

En este video, te mostramos cómo comenzar a operar con Interactive Brokers, una de las plataformas más completas y potentes del mercado. Aunque es un contenido que no aplicarás de inmediato en tu formación, servirá como referencia valiosa para cuando estés preparado para operar de forma independiente y profesional.

A través de una cuenta demo, se explica paso a paso cómo configurar el entorno de trabajo, personalizar gráficos y ejecutar órdenes de forma eficiente.

Puntos Clave:

- Características de Interactive Brokers: Plataforma avanzada con múltiples herramientas para operar acciones, CFDs, divisas, futuros y más.
- Creación y personalización de espacios de trabajo: Cómo añadir ventanas, herramientas de análisis, listas de seguimiento, noticias e indicadores.
- Configuración de gráficos: Ajuste de temporalidades, estilos visuales, colores y elementos técnicos como medias móviles o paneles de órdenes.
- Ejecución de órdenes: Cómo comprar y vender activos, calcular tamaños de posición, añadir stop loss y take profit, y gestionar operaciones abiertas.
- Diferencias entre acciones al contado y CFDs: Qué opción elegir según tu estrategia (inversión a largo plazo o trading activo).

Este vídeo te ayudará a familiarizarte con el entorno de Interactive Brokers y sentará las bases para que puedas sacarle el máximo provecho cuando llegue el momento de operar de forma real y estructurada.
